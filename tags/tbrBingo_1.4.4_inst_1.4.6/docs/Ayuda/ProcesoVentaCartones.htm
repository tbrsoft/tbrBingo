Proceso de Venta de Cartones.

El proceso de venta de cartones de un bingo consta de los siguientes pasos:

  1. Registre un Nuevo Bingo.
  2. Abra el Bingo.
  3. Asigne los cartones a los vendedores.

Aqui puede optar por dos modos diferentes de carga de las ventas, el modo manual y el automático.

Modo Manual:

Una vez asignados los cartones a un vendedor, el mismo procede a vender dichos cartones. Para registrar las ventas ingrese a la Consulta de Cartones. Una vez ubicado en la consulta, puede registrar la venta siguiendo el procedimiento detallado aquí.

Modo Automático:

Para utilizar este modo debe ingresar a la Consulta de Cartones.

Una vez ubicado en dicho formulario, debe filtrar loas cartones por vendedor, seleccionando el mismo de la lista desplegable. Posteriormente seleccione la opción Exportar. El sistema le solicitará la ruta donde desea guardar el archivo generado. Una vez ingresada dicha ruta y guardado el archivo, debe entregar el mismo al vendedor asignado.

El vendedor contará con un software adjunto al presente, cuya función es la registración de las ventas por parte de los vendedores. En esta etapa el vendedor procede a la distribución de los cartones y registro de las ventas en dicho programa.

Una vez que el vendedor haya registrado las ventas, el software le permite generar un archivo con el registro de las mismas. El vendedor debe enviarle dicho archivo para poder ingresar los datos al programa.

Cuando el vendedor le entregue el archivo de registro de ventas, debe abrir el Bingo correspondiente, ingresar a la consulta de cartones y seleccionar la opción Importar. El sistema toma los datos del archivo y los incorpora a la Base de Datos.
