Nuevo Bingo

En esta sección se explica como registrar un nuevo Bingo.

Para crear un bingo debe ingresar a la Consulta de Bingos, y seleccionar la opción Nuevo Bingo.

A continuación el sistema muestra la siguiente pantalla:

 
La pantalla permite asignar un nombre al bingo para poder identificarlo en la lista. También permite definir la cantidad de sorteos del bingo y los premios asociados a cada sorteo a través de la opción Definir Premios.

Además puede seleccionar el modo de Extracción de Bolillas, que puede ser manual o automático.

Una vez ingresados los datos se elige la opción Aceptar y el sistema registra el bingo.

Es importante recalcar que cuando se guarda el bingo el sistema genera automáticamente los cartones según la cantidad ingresada por el usuario.
