Realizar Sorteo

El proceso para realizar el sorteo de un Bingo es el siguiente:

  1. Abrir el Bingo.
  2. En el menú Bingo, seleccione la opción Realizar Sorteo.
  3. El sistema muestra la pantalla de Sorteo, donde se muestra una lista de los sorteos registrados. Para cada sorteo se muestra además el estado y si el sorteo ya fué realizado, muestra la cantidad de ganadores. En la lista de sorteos, puede seleccionar un elemento para ver en la parte inferior una lista de ganadores si corresponde.
  4. Para cada sorteo registrado se debe elegir el botón Realizar Siguiente Sorteo.
  5. El sistema muestra la pantalla de sorteo, donde se permite ingresar los números si el bingo fué configurado como manual, o bien generados automáticamente si el sorteo es definido como automático.
  6. A medida que se van generando o ingresado los números, el sistema controla si hay ganadores. En caso de haber ganadores, el sistema detiene el sorteo, muestra el botón Cerrar, envía los datos al formulario general de sorteo y muestra los ganadores en la lista inferior. Una vez realizado el sorteo, debe cerrar el mismo y repetir a partir del paso 4 para cada sorteo.

Los datos de los sorteos son almacenados y pueden ser consultados a través de la opción Consultar Resultados Sorteo, dentro del menú Bingo. Dicha opción se muestra una vez finalizados todos los sorteos de un bingo en lugar de la opción Realizar Sorteo.

En caso de que el sorteo deba mostrarse al público, el sistema cuenta con una pantalla se sorteo extendida. Dicha pantalla se muestra si se selecciona la opción Extender Pantalla.
