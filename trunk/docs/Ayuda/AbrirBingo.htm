Abrir Bingo

Para poder utilizar las opciones relacionadas a un bingo en particular (como puede ser realizar el sorteo de un bingo, ver los cartones del mismo, registrar la venta de los cartones, etc.) debe abrir un bingo previamente.

Para ello, seleccione un bingo de la lista en la Consulta de Bingos, y posteriormente la opción Abrir.

El sistema le informará que el bingo seleccionado se establecerá como activo, y se mostrará el nombre del mismo en la barra de estado del formulario principal.

Una vez abierto el Bingo, quedan disponibles las siguientes acciones asociadas a un bingo:
  1. Realizar el Sorteo del Bingo.
  2. Consultar el listado de Cartones.
  3. Asignar Cartones a Vendedores.
