Activación del Software

El presente software necesita un archivo de licencia para funcionar. El fin de dicha licencia es para evitar la piratería.

La activación del sistema consiste en tres pasos:

1. Generar archivo de información de la PC: El sistema obtiene información única de la PC donde se instalará el software. (Menú Ayuda -> Acerca de... Dentro del formulario utilizar botón "Generar archivo de licencia")
2. Enviar este archivo a tbrSoft, que analizará los datos y generara un fichero de licencia para uso exclusivo de la PC desde la cual fue generado el primer archivo.
3. Insertar el fichero de licencia. tbrSoft le enviará un archivo de licencia, que una vez incluido, activa el sistema para su uso. (Menú Ayuda -> Acerca de...  Dentro del formulario, botón "Insertar Licencia").

Una vez activada la licencia, podrá comenzar a utilizar el software sin inconvenientes.
