tbrBingo


Bienvenido a la ayuda de tbrBingo. Aquí encontrará toda la información necesaria para poder utilizar el software.

Al iniciar el programa se encontrará con las siguientes opciones:
  1. Bingos: Donde podrá consultar la lista de bingos registrados.
  2. Vendedores: Aquí puede administrar los datos de los vendedores.

Puede consultar esta ayuda presionando la tecla F1 desde cualquier parte del programa.
