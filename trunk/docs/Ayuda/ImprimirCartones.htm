Imprimir Cartones

Para imprimir los cartones siga los siguientes pasos:

  1. Abra el bingo. (Consulte como se abre un bingo aquí).
  2. El bingo debe contar con un diseño del mismo. Para diseñar un formulario de impresión de cartón, consulte aquí.
  3. Una vez abierto el bingo y diseñado el cartón, debe elegir la opción Imprimir Cartones que se encuentra en el menú Bingo.

 

El sistema muestra el formulario de impresión de cartones. Debe colocar los formularios preimpresos en la impresora, y mediante la opción imprimir comienza el proceso de impresión.

El usuario cuenta con la posibilidad de pausar la impresión para solucionar cualquier situación que pueda presentarse durante el proceso.

El sistema indica el estado de impresión de cada cartón y muestra el avance del proceso en la lista.
