Registrar Venta de Cartones.

Para registrar la venta de cartones el sistema cuenta con la siguiente pantalla.

 

En la misma se muestran los datos del cartón junto con los del vendedor. Para registrar los datos del comprador puede completar los campos en la sección Comprador si es un nuevo comprador, o ingresar partes del nombre o apellido y seleccionarlo en la lista si el comprador ya se encuentra en la base de datos.

Una vez ingresados los datos, seleccione Aceptar para registrar la venta.
